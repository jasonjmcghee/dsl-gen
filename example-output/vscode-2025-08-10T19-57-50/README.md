# calculator language Syntax Highlighting

This extension provides syntax highlighting for calculator language files (`.calc`).

## Features

- Syntax highlighting for calculator language
- Automatic bracket matching
- Comment toggling support

## Installation

1. Copy this folder to your VSCode extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions`
   - macOS/Linux: `~/.vscode/extensions`
2. Restart VSCode
3. Open any `.calc` file to see syntax highlighting

## Building from source

```bash
# Package the extension
npx vsce package

# Install the generated .vsix file
code --install-extension calc-syntax-*.vsix
```
